*---7 y media---*

*DESCRIPCION:*    
El proyecto es una version del juego clasico español de las 7 y media, similar al blackjack, el juego se divide en dos partes en la primera el jugador pueda pedir cartas al crupier hasta que el quiera o hasta que la suma de sus puntuaciones supere los 7.5 puntos, cada carta tiene su propio valor a excepcion de las cartas de figura (Sota,Caballo y Rey o su variante francesa J, Q y K) estas cartas valdran 0.5 puntos

Todo esto sobre una interfaz sencilla pero agradable que le permite al usuario tener una experiencia agradable 

*RELEASES:*    
V 1.0 - Juego completamente jugable del clasico juego español de las 7 y media.
V 2.0 - Juego completo con el añadidio de cambiar de baraja entre la española y la baraja francesa

*LENGUAJES:*  
HTML: Usado para crear las visuales del proyecto
CSS: Usado para dar forma y aspecto a los objetos del html
JAVASCRIPT: Usado para hacer la logica del juego

*INSTALACIÓN Y CONFIGURACIÓN:*
